# CapStation Brand Palette

Palette extracted from the uploaded CapStation logo.

## Core colors

| Token | Hex | Use |
|---|---:|---|
| `--brand-primary` | `#007038` | Main brand green, buttons, icons, active states |
| `--brand-primary-dark` | `#005F2E` | Header, footer, dark gradients, hover states |
| `--brand-primary-deep` | `#003E21` | Premium dark background, overlays |
| `--brand-primary-light` | `#009A4B` | Highlights, progress, animated lines |
| `--brand-secondary` | `#D8A301` | Gold accent, CTA highlights, rewards |
| `--brand-secondary-dark` | `#B78300` | Gold hover, deep shadows |
| `--brand-olive` | `#629C2D` | Secondary visual accents |
| `--brand-black` | `#050807` | Text on light backgrounds / luxury dark mode |
| `--brand-white` | `#FFFFFF` | Main light surfaces |
| `--brand-cream` | `#FFF8E6` | Warm background sections |
| `--brand-muted` | `#6B7280` | Muted text |
| `--brand-border` | `rgba(0, 112, 56, 0.14)` | Soft borders |

## Visual direction

The brand should feel:
- Saudi / Gulf market friendly.
- Premium, clean, trustworthy.
- Driver-first and growth-focused.
- Technology-enabled, not a normal static landing page.
- Modern transport platform with road, route, city, captain, and motion visuals.
