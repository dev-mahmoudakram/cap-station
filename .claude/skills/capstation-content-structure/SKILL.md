# Skill: CapStation Content Structure

Use this skill whenever writing or organizing landing page content.

## Content positioning

CapStation is not just a delivery app. It is a captain support and operations platform.

Primary audience:
- Captains/drivers.

Secondary audience:
- Companies and partners that need captain operations, delivery, transport, or driver network support.

## Messaging hierarchy

1. CapStation helps captains access more suitable opportunities.
2. It improves driver workflow, income visibility, and support.
3. It gives partners better access and operational control.
4. It is smart, local, and built for the Saudi/Gulf market.

## Homepage sections and content intent

### Hero
Make the user understand the product in 5 seconds.

### Problem
Show real driver pain:
- scattered apps
- wasted time
- unclear opportunities
- poor income visibility
- support issues

### Solution
CapStation brings structure, opportunity, and support.

### How it works
Simple steps from registration to earning.

### Captain features
Focus on practical benefits.

### Partner features
Focus on reliability and operations.

### Journey timeline
Turn the captain's journey into a visual story.

### Stats
Use placeholder metrics until real data is available.

### Testimonials
Realistic, short, benefit-focused.

### FAQ
Answer objections clearly.

### CTA
Make action obvious.
