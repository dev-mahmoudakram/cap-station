# CapStation Claude Code Pack

This pack contains a master prompt, project instructions, and reusable Claude Code skill files for rebuilding and rebranding the Fazza website into the new **CapStation** identity.

## Target stack

- Laravel 13
- Blade components
- Bootstrap
- SCSS
- Vite
- GSAP + ScrollTrigger for interactive motion
- Arabic as default language
- English localization
- RTL/LTR support
- SEO, performance, accessibility, and responsive design

## Suggested usage

1. Copy the `.claude` folder and `CLAUDE.md` into the project root.
2. Open the Laravel project in Claude Code.
3. Paste `MASTER_PROMPT.md` into Claude Code.
4. Ask Claude to work phase by phase, not all at once.
5. Review every generated file before running migrations or installing packages.

## Important

The website should be Arabic-first. English is secondary. Avoid Tailwind. Use Bootstrap + SCSS only.
